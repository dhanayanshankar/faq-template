  • Basics

  • How do I edit the text?

    In case you want to add new content, please place the cursor at the required place and start typing, Similarly for deletion, use the delete key or backspace key. If the area is not editable, please add a comment and mention your changes. "

  • How to copy the text?

    There are several shortcut keys available in Editgenie, similar to the ones available in word. Cut : Ctrl + X Copy : Ctrl + C Paste : Ctrl + V Undo : Ctrl + Z Redo : Ctrl + Y Find : Ctrl + F Find & Replace : Ctrl + H Bold : Ctrl + B Italic : Ctrl + I Underline : Ctrl + U

  • How do I italicize text?

    Select the text you wish to format and use the following shortcuts to do or undo the format. Bold : Ctrl + b , Italic : Ctrl + i , Underline : Ctrl + u"

  • How to format text bold?

    "Select the text you wish to format and use the following shortcuts to do or undo the format. Bold : Ctrl + b , Italic : Ctrl + i , Underline : Ctrl + u" .

  • How do i add a comment?

    For adding a comment on the text, which is non-editable, you can right-click on the area where you need changes and choose "comment" option and mention your changes. You can also use the option "Insert comments" available on the main menu bar. .

  • How do i find a word ?

    A word can be found using the 'Find' button available in the toolbar. 'Ctrl+F' keyboard shortcut can also be used. .

  • How do i replace a word ?

    A word can be found and can be replaced with the new word with 'Find and replace' option available in toolbar. 'Ctrl+H' keyboard shortcut can also be used. 'Replace' or 'Replace all' options are used to replace single or multiple occurrence. .

  • How do i upload a new file ?

    Files can be attached or uploaded using the clip button present on Menu bar. Click on the button and browse the files to be attached. You can add your comments in the description column. Same button can be used to upload revised or new graphic files also.

  • Front Matter

  • How do i add a new ORCID?

    Place the cursor on the author's name and right-click to add a new ORCID.

  • How do i correct invalid ORCID?

    When you add a orcid num, it gets validated against the orcid database. If it mismatches with the author names, you get ORCID invalid error. You can delete and add the correct ORCID num again.

  • How to delete orcid ?

    Right click on the author name and you will find an option to delete the orcid ?

  • How to correct Author name ?

    EditGenie does not provide an option to edit the author's name. Add a comment on the author group and mention the new name.

  • How to delete author ?

    Editgenie does not allow you to delete a author. You can add a comment on the author for deletion.

  • How to remove/edit the email address of corresponding author ?

    EditGenie does not provide an option to edit the author's details. Add a comment on the author group and mention the new email address ?

  • How do i add affiliation ?

    EditGenie does not support adding a new affiliation. You can add a comment on the author and provide the new affiliation content. .

  • How do i correct the author affiliation ?

    EditGenie does not support editing affiliation. You can add a comment to the author and provide the changes.

  • How to include the new keywords?

    Double click on the keyword section. This will open a form where you can add a new one using the + button .

  • How do I edit article title?

    EditGenie does not support editing article titles. You can add a comment on the title and provide the changes.

  • How do I add biography to author?

    EditGenie does not allow to add biography. You can add a comment on author name using right click and provide the biography content. You can also attach a file containing biography using the clip button on menu bar.

    Please double click to open the queries displayed in red on the page and answer them. Queries can be answered in sequence by moving to the next query by clicking the 'Next' button. If the correction/change is directly made in the article, please click the checkbox to mark it as resolved.
